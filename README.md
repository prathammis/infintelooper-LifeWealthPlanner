# LifeWealth Planner

Interactive hackathon starter for financial life planning.

## What’s in the scaffold

- React 18 + Vite + TypeScript app shell
- Monthly projection engine with inflation and drawdown handling
- Zoomable SVG timeline with drag-adjustable milestones
- Family mode with split contributions
- Template presets for fast demo onboarding
- Smart goal parser for quick milestone creation
- Export actions for SVG, PNG, and a print-friendly plan summary

## Run it

1. Install dependencies: `npm install`
2. Start the app: `npm run dev`
3. Build for production: `npm run build`

## Four-step execution plan

1. Lock the default onboarding flow and template import.
2. Let users add and drag milestones on the timeline.
3. Keep sliders, family shares, and shortfall messaging live.
4. Finish the share/export path so the demo ends with a clear artifact.
